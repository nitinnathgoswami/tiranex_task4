# 🛒 Retail Sales Data Analysis & Prediction

## 📌 Project Overview

This project focuses on cleaning, processing, analyzing, and predicting retail sales data to extract meaningful business insights and understand sales trends.

---

## 🧠 Key Features

* Data cleaning (handling missing values, formatting issues)
* Exploratory Data Analysis (EDA) using Python
* Visualization of sales, profit, and trends
* Machine learning model for sales prediction

---

## 📊 Tools Used

* Python (Pandas, NumPy)
* Matplotlib, Seaborn
* Scikit-learn
* Jupyter Notebook

---

## 📈 Key Insights

* Sales vary significantly across different regions
* Certain product categories generate higher profits
* Discounts impact overall sales and profitability
* Monthly sales show seasonal trends
* Some customer segments contribute more to revenue

---

## 🤖 Machine Learning Part

* Model Used: Linear Regression
* Features: Discount, Quantity
* Target: Sales
* Evaluation: MAE, MSE, R² Score

---

## 🎯 Outcome

This project demonstrates data cleaning, visualization, and machine learning skills by analyzing retail sales data and predicting future sales trends.
